# 2장. 하드웨어 구성

XLeRobot의 물리적 구성 요소와 하드웨어 설계를 학습합니다.

## 📚 이 장에서 배울 내용

- 로봇 제작에 필요한 부품 목록
- SO-100/SO-101 로봇 팔의 구조와 동작 원리
- 모바일 베이스와 옴니휠 시스템
- 카메라 및 센서 구성
- 3D 프린팅 가이드
- 전체 조립 과정

## 🎯 학습 목표

이 장을 완료하면:
- ✅ 로봇 제작에 필요한 모든 부품을 파악할 수 있습니다
- ✅ 각 하드웨어 컴포넌트의 역할을 이해할 수 있습니다
- ✅ 3D 프린팅 파일을 준비하고 출력할 수 있습니다
- ✅ 로봇을 처음부터 조립할 수 있습니다

## 💡 학습 전 참고사항

- 이 장은 **실제 로봇을 제작하려는 경우**에 중요합니다
- **시뮬레이션만 사용**한다면 3장으로 건너뛸 수 있습니다
- 하드웨어 지식은 시뮬레이션 이해에도 도움이 됩니다

## 🔧 필요한 사전 지식

- 기본적인 기계 조립 능력
- 3D 프린팅 기초 (또는 서비스 이용)
- 전자 부품 다루기 기초
- 공구 사용법
