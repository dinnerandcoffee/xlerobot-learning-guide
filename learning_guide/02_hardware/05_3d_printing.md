# 2.5 3D 프린팅 가이드

XLeRobot의 모든 구조 부품은 3D 프린터로 제작할 수 있습니다. 이 섹션에서는 프린팅 설정과 팁을 다룹니다.

## 1. 필요한 3D 프린팅 파일

### XLeRobot_0_3_0.3mf (통합 파일)

**포함 부품:**
- SO-101 팔 부품 (좌/우 각 세트)
- 소프트 핑거 (TPU95A)
- 베이스 플레이트 (상/하단)
- 옴니휠 마운트
- 헤드 짐벌 부품
- 케이블 라우팅 부품

**파일 위치:**
```
hardware/XLeRobot_0_3_0.3mf
```

### 개별 STL 파일

```
hardware/
├── Ender_Follower_SO101.stl       # 하드 그리퍼 핑거
├── SO101_soft_fin.stl             # 소프트 핑거 (TPU)
├── Gimbal_mesh_all_d435.stl       # D435 짐벌
├── Gimbal_mesh_all_d455.stl       # D455 짐벌
└── step/                          # STEP 파일 (수정용)
    ├── Drive_Motor_Mount_upgraded.step
    ├── soft_gripper_finger.step
    └── RGBD_Gimbal/
```

## 2. 프린터 요구 사항

### 최소 사양

- **빌드 볼륨**: 220 × 220 × 250mm
- **노즐**: 0.4mm (권장)
- **압출기**: 다이렉트 드라이브 (TPU용)
- **베드**: 가열 베드 필수

### 권장 프린터

- **Ender 3 V2/V3**: $200~300, 입문용
- **Prusa MK4**: $700+, 신뢰성
- **Bambu Lab P1S**: $600+, 속도

---

## 3. 재질별 프린팅 설정

### PLA (구조 부품용)

| 설정 | 값 | 비고 |
|------|-----|------|
| 노즐 온도 | 200-210°C | 브랜드별 조정 |
| 베드 온도 | 60°C | 첫 층 밀착 |
| 프린트 속도 | 50-60mm/s | 표준 |
| 레이어 높이 | 0.2mm | 품질/속도 균형 |
| 인필 | 20-30% | 충분한 강도 |
| 인필 패턴 | Gyroid | 등방성 강도 |
| 벽 두께 | 4줄 (1.6mm) | 강도 향상 |
| 서포트 | 필요 시 | 각도 45° 이상 |

### PETG (내구성 부품용)

| 설정 | 값 | 비고 |
|------|-----|------|
| 노즐 온도 | 230-240°C | PLA보다 높음 |
| 베드 온도 | 80°C | 뒤틀림 방지 |
| 프린트 속도 | 40-50mm/s | 느리게 |
| 레이어 높이 | 0.2mm | |
| 인필 | 30-40% | PLA보다 높게 |
| 쿨링 팬 | 50% | 과냉각 금지 |

### TPU95A (소프트 핑거용)

| 설정 | 값 | 비고 |
|------|-----|------|
| 노즐 온도 | 220-230°C | Shore 95A |
| 베드 온도 | 60°C | |
| 프린트 속도 | **20-30mm/s** | 매우 느리게! |
| 레이어 높이 | 0.15-0.2mm | |
| 인필 | 20-30% | 너무 높으면 딱딱함 |
| 리트랙션 | 최소화 (1mm) | TPU 특성 |
| 쿨링 팬 | 30-50% | |

**TPU 팁:**
- 다이렉트 드라이브 압출기 필수
- 필라멘트 경로 최소화
- 첫 층 속도 10mm/s
- 베드 접착제 사용 (스틱 glue)

## 5. 프린팅 방향 최적화

### SO101 팔 부품

```
올바른 방향:
  ┌─────┐
  │     │  ← 평평한 면을 베드에
  └─────┘

잘못된 방향:
    │
    │  ← 서포트 많이 필요
    │
```

**원칙:**
- 큰 평면을 베드에 배치
- 서포트 최소화
- 레이어 방향 = 하중 방향

---

## 6. 서포트 전략

### 서포트가 필요한 부품

- 팔꿈치 조인트 (45° 이상 오버행)
- 짐벌 틸트 마운트
- 케이블 라우팅 홀

### 서포트 설정 (Cura)

```
Support Type: Tree
Support Overhang Angle: 50°
Support Density: 15%
Support Z Distance: 0.2mm
```

### 서포트 제거 팁

- 플러시 커터 사용
- 서포트 인터페이스 면 샌딩
- 필요 시 드릴로 구멍 정리

---

## 7. 후처리

### 필수 후처리

1. **서포트 제거**: 플러시 커터
2. **구멍 리밍**: 3mm 드릴 비트
3. **표면 샌딩**: 400방 사포
4. **나사 구멍 탭핑**: M3 탭

### 선택 후처리

- **아세톤 증기**: ABS만 (PLA 불가)
- **에폭시 코팅**: 방수, 강도 향상
- **페인팅**: 프라이머 + 스프레이

---

## 8. 품질 문제 해결

### 문제 1: 첫 층이 떨어짐

**원인:** 베드 레벨링 불량  
**해결:**
- 베드 레벨링 다시 실행
- Z 오프셋 조정 (-0.05mm)
- 베드 온도 +5°C

### 문제 2: 레이어 분리

**원인:** 온도 너무 낮음  
**해결:**
- 노즐 온도 +10°C
- 쿨링 팬 감소
- 인클로저 사용

### 문제 3: TPU 뭉침

**원인:** 리트랙션 과다  
**해결:**
- 리트랙션 거리 1mm 이하
- 프린트 속도 20mm/s
- 다이렉트 드라이브 확인

## 9. 프린팅 체크리스트

**프린트 시작 전:**
- [ ] .3mf 파일 슬라이서에 로드
- [ ] 재질 확인 (PLA/PETG/TPU)
- [ ] 프린터 베드 레벨링
- [ ] 노즐 클리닝
- [ ] 필라멘트 충분량 확인

**프린트 중:**
- [ ] 첫 층 밀착 확인 (5분 이내)
- [ ] 중간 점검 (4시간마다)
- [ ] 필라멘트 엉킴 확인

**프린트 완료 후:**
- [ ] 베드 식힘 (5분)
- [ ] 부품 제거
- [ ] 서포트 제거
- [ ] 구멍 리밍
- [ ] 치수 확인
